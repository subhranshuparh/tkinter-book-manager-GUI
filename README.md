# 📚 Book Management System (Tkinter + SQLite)

A simple desktop application built with **Python Tkinter** and **SQLite3** for managing a collection of books. The application provides a user-friendly graphical interface to perform CRUD (Create, Read, Update, Delete) operations on book records.

---

## 🚀 Features

- ➕ Add new books
- 🔍 Search books by Title, Author, Year, or ISBN
- ✏️ Update existing book records
- ❌ Delete selected books
- 📋 Display all books
- 🧹 Clear input fields
- 💾 Local database using SQLite3
- 🖥️ Simple and responsive Tkinter GUI

---

## 🛠️ Technologies Used

- Python 3
- Tkinter
- SQLite3

## 📦 Installation

### Clone the repository

```bash
git clone https://github.com/subhranshuparh/tkinter-book-manager-GUI.git
```

Move into the project folder

```bash
cd book-management-system-tkinter-sqlite
```

Run the application

```bash
python frontend.py
```

---

## 💾 Database

The application uses **SQLite3**, so no external database server is required.

The database file (`books.db`) is automatically created when the application runs for the first time.
